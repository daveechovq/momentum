# Momentum Brand Guide

## Brand Essence

**Momentum** is a premium team experience offering that creates genuine connection through voice, movement, and music. The brand conveys warmth, credibility, and transformation — sophisticated but approachable, confident but not corporate.

---

## Logo

### Primary Logo
- **Font:** Bodoni Moda (Italic, 500 weight)
- **Style:** Lowercase, italic
- **Spelling:** *momentum*

### Usage
- Always use lowercase italic styling
- Minimum clear space: Height of the "m" on all sides
- Never stretch, distort, or change the font weight

### Logo Colors
| Context | Color |
|---------|-------|
| On dark backgrounds | Warm White (#FDFCFA) |
| On light backgrounds | Charcoal (#2C2C2C) |
| Accent use | Terracotta (#C4785A) |

| Role | Font | Weights |

| Element | Size | Font | Weight |

- *Italics* in headlines use the accent color (usually Terracotta or Gold)
- The word "Momentum" when used in body copy should be styled with Bodoni Moda italic

## Visual Elements

### Grain Texture
A subtle noise texture overlays the entire site at 2.5% opacity, adding warmth and tactile quality.

### Gradients
Hero and accent sections use multi-stop gradients combining Forest, Forest Light, and Terracotta Deep:
```css
background: linear-gradient(165deg, #2D4A3E 0%, #3D5F50 50%, #A65D42 100%);
```

### Border Radius
| Element | Radius |
|---------|--------|
| Buttons | 100px (pill shape) |
| Cards | 16-24px |
| Icons/Badges | 50% (circle) or 8-16px |

## Tone of Voice

### Characteristics
- **Warm but confident** — Not salesy, not timid
- **Direct and honest** — "Most team-building is forgettable. This isn't."
- **Conversational** — "Here's what we've noticed..."
- **Human** — Real language, not corporate jargon

### Do
- Speak directly to the reader
- Acknowledge real problems
- Use contractions (we'll, you're, isn't)
- Let personality come through

### Don't
- Use buzzwords (synergy, leverage, optimize)
- Over-promise or use superlatives
- Sound like a press release
- Be preachy or condescending

### Example Phrases
✓ "Most team-building is forgettable. This isn't."
✓ "Real connection isn't talked about — it's experienced."
✓ "Three Specialists. One Vision."
✓ "Let's have a conversation."

✗ "Synergize your team's potential"
✗ "Best-in-class corporate wellness solutions"
✗ "Unlock unprecedented performance gains"
